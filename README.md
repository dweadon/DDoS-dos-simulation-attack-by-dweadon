DDoS Visual Simulator v1.4

A terminal-based Python application that visually simulates a DDoS attack using animations and progress indicators. This project is for educational and demonstration purposes only.

Features
Animated progress bar and packet counter
Randomized burst events for realistic effect
Customizable target and packet count (display only)
Loading animations and terminal UI effects
Cross-platform support
Disclaimer

This tool does not perform any real network activity. It does not send packets, connect to servers, or execute attacks. Everything is purely visual and simulated.

Requirements
Python 3.x
No external libraries required
Usage
python script.py

Follow the prompts to enter a target name and packet count. The simulation will then run in the terminal.

Notes
On Windows, the script may open a new command prompt window for better display
The “target” is only for visual purposes
Simulation behavior includes random elements for realism
